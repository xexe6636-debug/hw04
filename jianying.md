# 剪映声音克隆说明（任务二）

## 配音脚本来源

使用任务一生成的文章《人工智能正在悄悄重塑我们的学习方式》全文作为配音脚本。

---

## 操作步骤概要

> **说明**：以下步骤基于 **剪映专业版（PC 端）**，版本 ≥ 5.x；手机版"剪映 App"操作入口略有差异，但流程相同。

### 第一步：录制声音样本

1. 打开剪映专业版，点击顶部菜单 **"文字" → "声音克隆"**（或在"我的"页面找到"声音克隆"入口）。  
2. 按照界面提示，朗读指定例句（通常需要朗读 1～3 段，共约 30 秒），确保环境安静、无杂音。  
3. 录制完成后，平台对声音样本进行建模（约需 1～5 分钟）。  
4. 建模完成后，在"我的音色"中可看到已克隆的专属音色。

### 第二步：生成配音

1. 新建项目，添加任意一段视频或空白轨道。  
2. 点击 **"文字" → "智能字幕" 或 "文字朗读"**，将任务一全文粘贴进文本框。  
3. 在"朗读音色"选择框中，切换至 **"我的音色"**，选择刚才克隆的声音。  
4. 点击"开始朗读"，等待生成完毕。  
5. 在时间轴上可看到已生成的配音片段。

### 第三步：导出

1. 点击右上角 **"导出"**。  
2. 若只需音频，选择 **导出格式：MP3 / WAV**（部分版本需先导出视频再用其他工具提取音轨）。  
3. 若直接导出视频，默认格式为 **MP4，分辨率 1080P**。

---

## 导出文件说明

| 项目 | 内容 |
|------|------|
| 文件名 | `tts_output.mp3`（或 `tts_output.mp4`） |
| 格式 | MP3（音频）/ MP4（视频） |
| 时长 | 约 3～4 分钟（450 字 × 约 0.5 秒/字） |
| 音色 | 声音克隆（本人声音）或剪映内置"温柔女声/磁性男声"（替代路径） |

## 替代路径说明（若无法完成克隆）

若因以下原因无法完成声音克隆：

- 设备不支持（如 iOS 低版本、旧款剪映）
- 网络环境限制导致建模失败
- 隐私顾虑不愿提交声纹数据

**替代方案**：使用剪映内置音色（无需录音，直接从"系统音色"库中选择），对同一文稿进行 TTS 朗读，导出音频。功能与声音克隆相同，区别仅在于音色为平台预设而非本人声音。本报告采用此替代路径，导出文件已放入仓库根目录（或见下方链接）。
